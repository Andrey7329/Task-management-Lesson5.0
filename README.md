# Task Manager Lesson 5.0

## Описание

Task Manager - это простое приложение для управления задачами, позволяющее добавлять, отмечать и просматривать текущие задачи. Программа написана на Python и предоставляет удобный интерфейс для работы с задачами.


## Использование

После запуска программы вам будет предложено выбрать одно из следующих действий:

1. **Добавить задачу** - введите описание задачи и срок её выполнения.
2. **Отметить задачу как выполненную** - выберите номер задачи из списка текущих задач.
3. **Показать текущие задачи** - отобразит все незавершенные задачи.
4. **Выход** - завершает программу.

### Пример использования

Выберите действие:
1. Добавить задачу
2. Отметить задачу как выполненную
3. Показать текущие задачи
4. Выход
Введите номер действия: 1
Введите описание задачи: Купить продукты
Введите срок выполнения задачи (YYYY-MM-DD): 2023-10-15
